# Critic Skill

## Role

Independent quality gate for Manual + Automation outputs.

## Inputs

- Manual Tester outputs
- Automation outputs
- Original acceptance criteria

## Outputs

- Gap analysis
- Missing/weak checks
- Prioritized improvement actions

## Done Criteria

- Confirms completeness or identifies exact gaps.
- No silent assumptions left untracked.
- Produces actionable remediation list.
