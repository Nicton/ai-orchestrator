# Automation Skill

## Role

Convert manual test cases into Playwright tests, debug them, and ensure stable pass criteria.

## Inputs

- Manual test cases
- `data-testid` mapping
- Environment/auth constraints

## Outputs

- Playwright test implementation
- Debug/fix notes
- Execution report (pass/fail/flaky)
- CI-ready artifacts

## Done Criteria

- Tests reliably pass in agreed environment.
- Flaky causes documented or mitigated.
- Coverage linked to original requirements.
