# Magic Filters — Умные фильтры

Magic Filters — сохранённые наборы фильтров с одним кликом применения. Позволяют быстро переключаться между часто используемыми комбинациями фильтров.

## Где доступны

> Уточнение Product (2026-06-12): Magic Filters — **системный механизм всех списковых страниц** (перевозки, заявки, инвойс-строки, FU, TR, слоты и т.д.). Страницы отличаются только доступным набором фильтров; поведение (сохранить набор → применить одним кликом) везде одинаковое.

## Типы Magic Filters

| Тип | Кто видит | Описание |
|-----|----------|---------|
| **Private** | Только создатель | Личный фильтр пользователя |
| **Shared** | Все пользователи аккаунта | Общий фильтр для команды |

## Как работает

```
1. Пользователь настраивает фильтры на странице
   (статус, перевозчик, дата, маршрут — любые комбинации)

2. Нажимает [Save as Magic Filter]
   → Вводит название (например: "Активные DHL этой недели")
   → Выбирает: Private или Shared

3. Magic Filter появляется в списке сохранённых фильтров

4. В следующий раз: клик по названию фильтра
   → Все сохранённые условия применяются мгновенно
```

## Управление

| Действие | Описание |
|----------|---------|
| Создать | Настроить фильтры → [Save as Magic Filter] |
| Применить | Клик по названию Magic Filter в списке |
| Редактировать | Изменить условия → перезаписать фильтр |
| Удалить | Кнопка удаления рядом с фильтром |
