# Внутренняя документация — Shiptify Backend

> Язык: русский. Аудитория: разработчики команды.
> Источники: CLAUDE.md, doc/, Confluence TD space, Jira.

## Что это за проект

**Shiptify Backend** — серверная часть TMS (Transport Management System) на Node.js.
Система управляет перевозками: от создания заявки до отслеживания груза и выставления счётов.

**Стек:**
- Node.js >= 24.4.0, CommonJS (require/module.exports)
- Express.js 4.x (HTTP API, порт 3013)
- PostgreSQL + Sequelize ORM v3.35.1 (~466 моделей)
- Redis + Kue (фоновые задачи, порт воркера 3018)
- Kafka (события для микросервисов)
- Elasticsearch (поиск)
- AWS S3 (файлы, логи интеграций)
