# Quote Strategy — Стратегии котировок

Новый модуль автоматизации выбора перевозчика через конкурентные механизмы торгов.

> Источник: слайд `2026 05 - QUOTE STRATEGY`

## 5 стратегий котировок

### 1. Standard Quote (стандартная, по умолчанию)
Классический RFQ: перевозчики отвечают своими ценами, Shipper выбирает вручную.

### 2. Buy-It-Now (целевая цена)
Shipper устанавливает целевую (максимальную) цену. Первый перевозчик, предложивший цену ≤ целевой, автоматически получает заказ.

### 3. Hybrid (слепой → открытый)
1. Фаза 1: целевая цена скрыта (перевозчики "вслепую")
2. Фаза 2: через N минут (настраивается) цель становится видима → поведение как Buy-It-Now

### 4. Live Reverse Auction (живой аукцион)
Перевозчики видят лучшую текущую цену и могут снижать свою ставку в реальном времени. Побеждает минимальная цена в момент окончания.

### 5. Automated Sealed Bid (запечатанная ставка)
Каждый перевозчик подаёт одну ставку, не видя других. Лучшая ставка выигрывает автоматически.

---

## Safety Net (защитный механизм)

Если условия стратегии не выполнены к дедлайну (Answer Before):
→ Автоматика **прекращается**
→ Возврат к ручному выбору победителя

- Новые колонки/тултипы с квотами
- Чекбокс: исключить перевозчиков, превысивших квоту
