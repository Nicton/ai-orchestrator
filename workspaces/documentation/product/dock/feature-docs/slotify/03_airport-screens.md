# Airport Screens — Экраны аэропортного типа

Airport Screens — это режим отображения Slotify, предназначенный для вывода на большие экраны непосредственно на складе или в зоне ожидания. Экраны отображают актуальный список машин в формате табло отправлений/прибытий.

## Назначение

Крупный экран на складе показывает водителям и сотрудникам:
- Какие машины ожидаются в ближайшее время
- Какие уже находятся на территории / на воротах
- Какие операции завершены

Не требует взаимодействия — только отображение с автоматическим обновлением.

## Три режима отображения

### Upcoming (Предстоящие)

Слоты, запланированные на ближайшие часы, которые ещё не прибыли.

| Что показывается | Описание |
|-----------------|----------|
| Время слота | Плановое время прибытия/убытия |
| Перевозчик | Название компании-перевозчика |
| Номер транспорта | Из metadata слота |
| Тип груза | Краткий список cargo |
| Ворота (Dock Door) | Если уже назначены |
| Статус | `new`, `driver_called` |

**Фильтрация:** слоты со статусом `new` или `driver_called`, с плановым временем в будущем.

### Delayed (Задержанные)

Слоты, чьё плановое время уже прошло, но операция не завершена.

| Что показывается | Описание |
|-----------------|----------|
| Время слота | Плановое время (в прошлом — выделено красным) |
| Задержка | Насколько минут/часов задержан |
| Перевозчик | Название компании |
| Статус | `on_site`, `at_dock`, `loading`/`unloading` |

**Фильтрация:** плановое время (`slots.time`) < текущее время, статус не является конечным (`loaded`, `unloaded`, `left_dock`, `left_site`, `canceled`, `no_show`).

### Completed (Завершённые)

Слоты, операции по которым завершены за текущий день.

| Что показывается | Описание |
|-----------------|----------|
| Время завершения | `real_date` из слота |
| Перевозчик | Название компании |
| Ворота | Dock Door |
| Статус | `loaded`, `unloaded`, `left_dock`, `left_site` |

**Фильтрация:** статус конечный, `real_date` в текущем дне.

## Автоматическое обновление

Экраны обновляются без перезагрузки страницы:

| Параметр | Значение |
|---------|----------|
| Интервал обновления | Каждые 30–60 секунд |
| Механизм | Периодический GET к `/api/v1/dock-slots` |
| Сортировка | По времени слота (по возрастанию) |

**Эндпоинт:** `GET /api/v1/dock-slots` — слоты для dock-вида с фильтрацией по локации и дате.

## Используемые поля слота

| Поле | Описание |
|------|----------|
| `slots.time` | Плановое время |
| `slots.real_date` | Фактическое время операции |
| `slots.status` | Текущий статус |
| `slots.dock_door_id` | Назначенные ворота |
| `slot_contents` | Список типов груза |
| Metadata | Номер транспорта, имя водителя |

## Статусы слота и экраны

| Статус (внутренний) | Публичный статус | Экран |
|--------------------|-----------------|-------|
| `new` | `planned` | Upcoming |
| `driver_called` | `planned` | Upcoming |
| `on_site` | `ongoing` | Delayed (если время прошло) |
| `at_dock` | `ongoing` | Delayed |
| `loading` / `unloading` | `ongoing` | Delayed |
| `loaded` / `unloaded` | `delivered` / `loaded` | Completed |
| `left_dock` | `delivered` | Completed |
| `left_site` | `delivered` | Completed |
| `no_show` | `no_show` | Completed (особый вид) |
| `canceled` | `cancelled` | Скрыт |

## Настройка и доступ

Экраны Airport Screens доступны через Planning интерфейс. Оператор может открыть экран в полноэкранном режиме для вывода на TV-панель на складе.

Фильтры применяются по:
- Конкретной локации (PML)
- Зоне (LocationZone)
- Направлению (Reception / Expedition)
