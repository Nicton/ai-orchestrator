## Фаза 1 — Фундамент (1–2 недели)

| # | Задача | Результат |
|---|--------|-----------|
| 1.1 | ✅ **Skill `30-code-truth-verifier`** (сделано 2026-06-11, коммит fb109b4) — «вопрос → код → вердикт с file:line» | Любой открытый вопрос проверяется по коду до эскалации; на входе списки вопросов, на выходе ответы + дефект-кандидаты |
| 1.2 | **Stage-timeout в DAG-воркере** (S7) | Пайплайнам можно доверять: зависший агент убивается, retry по конфигу |
| 1.3 | **Восстановить транскрипцию** (S8): подключить STT-провайдера вместо удалённого Whisper | Intake-пайплайн снова рабочий |

## Метрики успеха

- Покрытие RTM ≥ 98% (не деградирует) · Открытых вопросов «старше 2 недель» → 0
- Кейсы в Qase: 0 → 800+ (Фаза 2) · Время «вопрос → ответ из кода»: дни → минуты
- Дефекты, найденные аудитом до прода (сейчас 28) — трекать escape rate
