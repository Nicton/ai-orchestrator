## Страницы и экраны

### [Название страницы]

**URL:** `/route/path`
**Frontend:** `workspaces/frontend/public/app/module/`

#### Что видит пользователь

| Поле / Блок | Описание | Откуда берётся |
|-------------|---------|----------------|
| Поле 1 | Описание | `Model.field` |
| Статус | Draft / Pending / ... | `Model.status` |

#### Действия

| Действие | Что происходит | Условие |
|----------|---------------|---------|
| Кнопка "Создать" | Открывает wizard | Роль: api |
| Кнопка "Отменить" | Меняет статус | Только если Draft |

#### Переходы

- Клик по строке → `/route/{id}` (описание куда)
- Кнопка "Назад" → `/parent-route`

#### Мутации

**Внутренние (БД):**
- `Model.status` → `new_value`
- Создаётся запись `OtherModel`
- Очередь Kue: задача `taskName`

**Внешние (интеграции):**
- Carrier API: `POST /endpoint` — бронирование
- Email: `mailStatus` → уведомление shipper'у
- Webhook: событие `eventName`

**Backend service:** `app/services/service_name.js` → функция `functionName()`
