## Назначение

Vanna AI позволяет пользователям задавать вопросы о данных платформы на естественном языке — система автоматически конвертирует их в SQL-запросы и возвращает результат в виде таблицы.

**Пример запроса:** "Покажи топ 15 клиентов по количеству отправлений за последние 30 дней"

---

## Доступ и роли

Инструмент доступен **только внутренним командам**. Не виден shipperам и carriерам.

| Группа | Команда | Доступ |
|---|---|---|
| Sales | Отдел продаж | Да |
| Support | Служба поддержки | Да |
| Product | Продуктовая команда | Да |
| Project | Проектный офис | Да |
| Shippers / Carriers | Внешние пользователи | Нет |

## Функциональность

### Естественно-языковые запросы
Пользователь вводит вопрос в строку поиска. AI анализирует запрос, подбирает SQL и выполняет его. Результат отображается в таблице.

### Сохранённые избранные запросы
Каждая команда (Sales / Project / Support / Product) имеет собственный набор сохранённых запросов-избранных. Позволяет быстро повторно запускать часто используемые отчёты.

### Просмотр доступных полей
Встроенный браузер полей позволяет узнать, какие данные доступны для запросов, без написания SQL вручную.

### Экспорт
Результаты выгружаются в **CSV** или **XLS**.

## Авто-обогащение запросов

Vanna AI автоматически дополняет запросы по следующим правилам:

| Условие | Добавляемое правило |
|---|---|
| Сортировка не указана | `ORDER BY ... DESC` |
| Запрос подразумевает "список" | `TOP 15` (ограничение записей) |

---

## Известные проблемы

| Проблема | Описание |
|---|---|
| Фильтр по `shipper_id` | Непредсказуемо снижает количество записей в результатах |
| Пресет "booked shipments" | Возвращает 0 результатов: используется неправильный тип сущности в запросе |
| 30-дневное окно | Запросы с диапазоном 30 дней могут завершаться ошибкой |

## Техническое описание

Vanna AI — это отдельный SaaS/standalone сервис. В основном бэкенде Shiptify (`workspaces/backend/`) код интеграции с Vanna не найден. Инструмент подключается через embed (iframe) или redirect.

Бэкенд базы знаний Vanna обучен на схеме данных Shiptify и доступных полях для аналитических запросов.
